# Привет, меня зовут Андрей! 👋

### Frontend-разработчик из Липецка &nbsp;![Codewars](https://www.codewars.com/users/Tro48/badges/micro)

Отвечаю за фронтенд-направление на проекте: декомпозирую задачи для команды, наполняю бэклог, провожу code review и выстраиваю процессы тестирования (unit / component / e2e). Специализируюсь на **React / Next.js**, но регулярно работаю и с более низкоуровневыми вещами — **WebRTC**, **Canvas / WebGL**, **gRPC-Web**.

Параллельно веду собственный продукт — приложение для планирования встреч в дружеских компаниях (Next.js, NestJS, Supabase, Docker, сейчас в закрытой бете). Это дало опыт проектирования архитектуры «с нуля»: от auth и state machines до инфраструктуры.
